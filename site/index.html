README & User Guide

What does Lan Lens do?

Lan Lens is a macOS menu bar app that scans your local network to discover connected devices. It identifies device types, manufacturers, and services using multiple fingerprinting techniques.

Does Lan Lens send my data to external servers?

By default, no. Lan Lens works entirely offline using bundled fingerprint databases. The optional Fingerbank API integration (disabled by default) can be enabled for enhanced device identification.

Why does Lan Lens need network permissions?

Lan Lens needs network access to scan your local network and discover devices using ARP scanning, mDNS/Bonjour, and SSDP/UPnP protocols.

What is the API Server feature?

Lan Lens includes an optional local API server (disabled by default) that allows automation tools and scripts to query device information. This runs only on localhost (127.0.0.1) and is useful for home automation integrations.

How do I perform a deep scan?

Click on any device in the list, then click "Deep Scan" to perform port scanning and TLS fingerprinting for more detailed information about services running on the device.
