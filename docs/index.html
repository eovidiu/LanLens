Frequently Asked Questions

What does LanLens do?
LanLens is a macOS menu bar app that scans your local network to discover connected devices. It identifies device types, manufacturers, and services using multiple fingerprinting techniques.
Does LanLens send my data to external servers?
By default, no. LanLens works entirely offline using bundled fingerprint databases. The optional Fingerbank API integration (disabled by default) can be enabled for enhanced device identification, but requires you to provide your own API key.
Why does LanLens need network permissions?
LanLens needs network access to scan your local network and discover devices. It uses ARP scanning, mDNS/Bonjour, and SSDP/UPnP protocols to find devices on your network.
What is the API Server feature?
LanLens includes an optional local API server (disabled by default) that allows automation tools and scripts to query device information. This runs only on localhost (127.0.0.1) and is useful for home automation integrations.
How do I perform a deep scan?
Click on any device in the list, then click "Deep Scan" to perform port scanning and TLS fingerprinting. This provides more detailed information about the services running on the device.
Can I export the device list?
Yes! Go to Settings and use the Export feature to save your device list as JSON or CSV format.
